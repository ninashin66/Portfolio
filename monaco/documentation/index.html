About

Monaco is a minimalist jquery powered personal vCard template with fullscreen slideshow background, build-in lightbox, working contact form, latest tweets, support for browser history buttons (back / next), and more. Fullscreen background will adapt to any screen resolution. Included PSD source file allows you to customize this template to your needs.

  • jQuery Powered
  • Fullscreen Slideshow
  • Working Contact Form
  • Back / Next Browser History Support
  • Latest Tweets
  • Font Face Fonts
  • Valid XHTML Code
  • PSD Source Included
  • Documentation Included

CSS Styles

The project uses the following cascade of template styles (in order):

Filename Description
jquery.qtip.css default styles for qTip2 (overwritten in base.css)
nivo-slider.css default styles for Nivo slider (overwritten in page.css)
jquery.jscrollpane.css default styles for jScrollPane (overwritten in base.css)
supersized.css default styles for Supersized (overwritten in base.css)
fancybox/jquery.fancybox.css default styles for Fancybox
base.css base styles for entire project
page.css styles for subpages (about, services etc.)

All CSS files are located in /style directory.

Plugins

Contact form

This project includes a contact form ready for use.
All you need is to configure some constants located in file plugin/contact-form/config.php:

Required data

Constant Description
CONTACT_FORM_TO_NAME name of recipient eg. "My Company"
CONTACT_FORM_TO_EMAIL e-mail address of recipent eg. "mycompany@gmail.com"

Optional SMTP settings

Constant Description
CONTACT_FORM_SMTP_PORT your SMTP port eg. "587"
CONTACT_FORM_SMTP_HOST your SMTP host eg. "smtp.gmail.com"
CONTACT_FORM_SMTP_USER login to your e-mail account eg. "mycompany"
CONTACT_FORM_SMTP_PASSWORD password to your e-mail account
CONTACT_FORM_SMTP_SECURE secure connection protocol: "SSL" or "TLS"

Texts and messages

Constant Description
CONTACT_FORM_SUBJECT subject of e-mail message
CONTACT_FORM_MSG_INVALID_DATA_NAME text displayed when name is empty
CONTACT_FORM_MSG_INVALID_DATA_MAIL text displayed when e-mail address is invalid
CONTACT_FORM_MSG_INVALID_DATA_MESSAGE text displayed when message field is empty
CONTACT_FORM_LABEL_DATA_NAME label for name field
CONTACT_FORM_LABEL_DATA_MAIL label for e-mail field
CONTACT_FORM_LABEL_DATA_MESSAGE label for message field
CONTACT_FORM_SEND_MSG_OK text displayed when e-mail was sent successfully
CONTACT_FORM_SEND_MSG_ERROR text displayed when e-mail cannot be sent


Twitter user timeline

This project includes a twitter user timeline ready for use.
All you need is to configure some constants located in file plugin/twitter-user-timeline/config.php:

Required data

Constant Description
TWITTER_USER_TIMELINE_USER the screen name of the user for whom to return latest tweets
TWITTER_USER_TIMELINE_COUNT specifies the number of tweets to try and retrieve, up to a maximum of 200
TWITTER_USER_TIMELINE_CONSUMER_KEY consumer key
TWITTER_USER_TIMELINE_CONSUMER_SECRET consumer secret key
TWITTER_USER_TIMELINE_ACCESS_TOKEN access token
TWITTER_USER_TIMELINE_ACCESS_TOKEN_SECRET access secret token

FAQ

Installing template

To install this template upload all files from www/php or www/html folder into HTTP server.
It is highly recommended to use PHP version instead of HTML (if it is possible).
Please note that plugins Contact form and Twitter user timeline work only in PHP version.


Creating new page

To create new page, please follow these steps:

1. Open file /script/main.js and add new entry to the page object:

var page=
{
	....
	'new_page'		:
	{
		menuIndex	:	0,
		imageIndex	:	1
	}
};

where new_page is a hash name of created page, menuIndex is a position of element in top menu, imageIndex is a image index from backgroundImage object.

2. Create new file /page/my_page.html.

3. Open file index.php, find a section Menu and add new entry as below:

<!-- Menu -->
<ul class="no-list header-menu clear-fix">
	....
	<li>
		<a href="#!new_page">New page</a>
		<span></span>
	</li>
</ul>
<!-- /Menu -->

4.Open file index.php, find a section Content and add new entry as below:

<!-- Content -->
<div class="content">

	<ul class="page-list clear-fix">
		....
		<li class="page-list-item" id="new_page">
			<?php require_once('page/new_page.html'); ?>
		</li>	
	</ul>

</div>
<!-- /Content -->

Twitter user timeline

To configure twitter user timeline, please follow these steps:

1. Create and setup your Twitter Account Application. You can learn more about it here.

2. Open file plugin/twitter-user-timeline/config.php and setup all necessary data (more informations you can find in tab Plugins / Twitter User Timeline).


Changing Google Maps coordinates

To change Google Maps coordinates in contact page, open file /script/main.js and change values for option googleMap.coordinates which is available in options object. You can designate coordinates using this tool: http://www.birdtheme.org/useful/googletool.html. 

var options=
{
	....
	googleMap			:
	{
		coordinates		:	[48.856614,2.352222]
	}
};

Changing background images

All images of background slider are stored in backgroundImage object in file /script/main.js.
